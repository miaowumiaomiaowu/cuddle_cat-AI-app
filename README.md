# 暖猫 (Cuddle Cat) - AI心理疗愈与旅行记录应用

## 📝 项目介绍

暖猫(Cuddle Cat)是一款专注于心理疗愈和旅行记录的Flutter应用。通过AI驱动的猫咪伙伴，为用户提供专业级心理咨询服务，同时记录和分享美好的旅行时光。应用采用温暖的莫兰迪色系设计，营造舒适治愈的使用体验。

## 🌟 核心功能

### 1. 🐱 猫咪伙伴系统

#### 1.1 猫咪选择与认领
- **品种选择**: 提供5种精美猫咪品种（孟加拉猫、缅因猫、波斯猫、布偶猫、暹罗猫）
- **性格匹配**: 每只猫咪拥有独特的性格特点，影响对话风格和互动方式
- **个性化命名**: 用户可以为猫咪取名，建立深度情感连接
- **新手引导**: 完整的引导流程帮助用户选择最适合的猫咪伙伴

#### 1.2 简化互动系统
- **温柔抚触**: 通过触摸屏幕与猫咪进行情感互动
- **心情同步**: 猫咪会根据用户的情绪状态展现相应的表情和行为
- **陪伴模式**: 猫咪静静陪伴用户，提供情感支持

### 2. 🧠 AI心理疗愈系统

#### 2.1 专业级心理咨询
- **心理咨询师水平**: 基于专业心理学理论训练的AI模型
- **认知行为疗法**: 运用CBT技术帮助用户识别和改变负面思维模式
- **情绪调节指导**: 提供科学的情绪管理策略和技巧
- **危机干预**: 识别用户的心理危机信号，提供及时支持和建议

#### 2.2 智能对话功能
- **深度情感理解**: 准确识别用户的情绪状态和心理需求
- **个性化治疗方案**: 根据用户的具体情况制定个性化的心理支持计划
- **长期记忆系统**: 记录用户的心理历程，提供连贯的治疗体验
- **猫咪视角对话**: 通过可爱的猫咪形象传递专业的心理支持

#### 2.3 心理健康监测
- **情绪追踪**: 记录和分析用户的情绪变化趋势
- **压力评估**: 定期评估用户的压力水平和心理状态
- **健康报告**: 生成个性化的心理健康报告和建议
- **预警机制**: 及时发现心理健康风险，提供预防性干预

### 3. 🗺️ 增强旅行记录系统

#### 3.1 智能地图功能
- **精准定位打卡**: 基于GPS的高精度地理位置记录
- **智能路线规划**: AI推荐最佳旅行路线和景点
- **离线地图支持**: 支持离线下载地图，无网络也能使用
- **多地图源**: 支持Google Maps、高德地图等多种地图服务

#### 3.2 丰富记录功能
- **多媒体日记**: 支持照片、视频、音频和文字的综合记录
- **情绪标记**: 为每个地点标记当时的心情和感受
- **天气记录**: 自动记录当时的天气状况
- **同行者标记**: 记录旅行同伴信息

#### 3.3 回忆与分享
- **时光轴回顾**: 按时间线展示所有旅行记录
- **地图足迹**: 在地图上可视化展示旅行轨迹
- **精美分享卡片**: 自动生成美观的分享卡片
- **社交平台集成**: 一键分享到微信、微博、小红书等平台
- **旅行统计**: 详细的旅行数据统计和分析

### 4. 👤 用户系统

#### 4.1 注册与登录
- **多种登录方式**: 支持手机号、邮箱、微信、QQ等登录方式
- **安全验证**: 短信验证码、邮箱验证等多重安全保障
- **隐私保护**: 严格的用户隐私保护机制
- **账号同步**: 支持多设备数据同步

#### 4.2 云数据存储
- **实时同步**: 用户数据实时同步到云端
- **数据备份**: 自动备份重要数据，防止丢失
- **跨设备访问**: 在任何设备上都能访问个人数据
- **离线支持**: 支持离线使用，网络恢复后自动同步

### 5. 🎯 新手引导系统
- **欢迎流程**: 温馨的应用介绍和功能说明
- **猫咪选择指导**: 帮助用户选择最适合的猫咪伙伴
- **功能演示**: 交互式的功能演示和使用教程
- **个性化设置**: 根据用户偏好进行个性化配置

## 🛠️ 技术架构

### 前端技术栈
- **开发框架**: Flutter 3.x
- **状态管理**: Provider + ChangeNotifier
- **UI设计**: 莫兰迪色系 + 手绘彩铅风格
- **本地存储**: Hive + SharedPreferences
- **网络请求**: Dio + 拦截器
- **图片处理**: cached_network_image + image_picker

### 后端技术栈
- **云数据库**:
  - **免费方案**: Firebase Firestore (Google提供的免费额度)
  - **备选方案**: 阿里云数据库 (如需更大容量)
- **用户认证**: Firebase Authentication
- **文件存储**: Firebase Storage / 阿里云OSS
- **推送服务**: Firebase Cloud Messaging

### AI与地图服务
- **AI对话模型**:
  - OpenAI GPT-4 API (专业心理咨询训练)
  - 百度文心一言 API (备选方案)
- **地图服务**:
  - 高德地图 SDK (国内用户)
  - Google Maps API (海外用户)
- **定位服务**: GPS + 网络定位

### 数据安全与隐私
- **数据加密**: AES-256加密存储敏感数据
- **传输安全**: HTTPS + SSL证书
- **隐私保护**: 本地数据脱敏处理
- **备份机制**: 自动云端备份 + 本地缓存

## � 云服务成本分析

### 免费方案 (推荐起步阶段)
**Firebase免费额度 (Spark计划)**
- **Firestore数据库**: 1GB存储 + 50,000次读取/天 + 20,000次写入/天
- **Authentication**: 无限制用户认证
- **Storage**: 1GB文件存储 + 1GB/天下载流量
- **Cloud Functions**: 125,000次调用/月
- **预计支持用户**: 1000-5000活跃用户

### 付费升级方案
**Firebase Blaze计划 (按使用量付费)**
- **成本预估**: 月活1万用户约$20-50/月
- **阿里云备选**: 如需要国内服务器，月成本约¥200-500

### AI服务成本
- **OpenAI GPT-4**: 约$0.03/1K tokens (专业心理咨询)
- **百度文心一言**: 约¥0.008/1K tokens (成本更低的备选)
- **预计成本**: 1000活跃用户约$50-100/月
